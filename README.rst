Install
------------------------------------------------------------------------------

``learn_python_fire`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install learn-python-fire

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade learn-python-fire
